# Cetacean Acoustic Communication Analysis - Summary Report
# 鯨類音響コミュニケーション解析 - サマリーレポート

## Hypothesis Validation Results / 仮説検証結果

### 1. Encoding Structure (エンコーディング構造)

- **Sperm_Whale**: Shannon entropy = 7.32 bits
- **Humpback_Whale**: Shannon entropy = 4.47 bits
- **Killer_Whale**: Shannon entropy = 6.66 bits
- **Fin,_Finback_Whale**: Shannon entropy = 2.28 bits
- **Bottlenose_Dolphin**: Shannon entropy = 5.79 bits
- **Beluga,_White_Whale**: Shannon entropy = 6.19 bits

### 2. Nonlinear Coupling / Beat Frequency Effects (非線形結合 / うなり効果)

- **Sperm_Whale**: mean bicoherence = 0.000751
- **Humpback_Whale**: mean bicoherence = 0.000285
- **Killer_Whale**: mean bicoherence = 0.000131
- **Fin,_Finback_Whale**: mean bicoherence = 0.000124
- **Bottlenose_Dolphin**: mean bicoherence = 0.000340
- **Beluga,_White_Whale**: mean bicoherence = 0.000386

### 3. Spectral Features (スペクトル特徴)

- **Sperm_Whale**: centroid = 2309 Hz, dominant = 1047 Hz
- **Humpback_Whale**: centroid = 593 Hz, dominant = 508 Hz
- **Killer_Whale**: centroid = 1804 Hz, dominant = 1547 Hz
- **Fin,_Finback_Whale**: centroid = 38 Hz, dominant = 29 Hz
- **Bottlenose_Dolphin**: centroid = 4021 Hz, dominant = 1430 Hz
- **Beluga,_White_Whale**: centroid = 1251 Hz, dominant = 128 Hz

### 4. Temporal Structure (時間構造)

- **Sperm_Whale**: spectral flatness = 0.3713
- **Humpback_Whale**: spectral flatness = 0.0016
- **Killer_Whale**: spectral flatness = 0.2187
- **Fin,_Finback_Whale**: spectral flatness = 0.0000
- **Bottlenose_Dolphin**: spectral flatness = 0.1051
- **Beluga,_White_Whale**: spectral flatness = 0.2374

## Interpretation / 解釈

The analyses above provide quantitative measures that can be used to evaluate the hypotheses about cetacean communication encoding:

1. **Entropy analysis** shows whether vocalizations carry structured information (lower entropy = more structured = more encoded)
2. **Bispectral analysis** reveals nonlinear frequency coupling, which would support the beat frequency / parametric speaker hypothesis
3. **CDMA orthogonality** tests whether different species/individuals have separable spectral signatures, supporting the encoding hypothesis
4. **ICI analysis** for sperm whales shows temporal click patterns that could serve as identity codes
